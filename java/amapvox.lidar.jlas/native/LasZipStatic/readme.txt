Ce projet a été généré avec QtCreator.

Il permet de générer une librairie statique (fichier .a ) à partir des sources de LasZip hébergées sur Github à l'adresse "https://github.com/LAStools/LAStools/tree/master/LASzip"

Cette librairie statique peut ensuite être utilisée dans le projet LasZipLibrary et évite ainsi de recompiler les sources à chaque changement.

Le fichier de sortie est nommé libLasZip.a
